# Preview Project Workflow

Follow these steps to run the local development server and preview the web application.

## Steps

1. **Start Development Server**
   Start the local Next.js development server (using turbopack and port 9002 by default):
   ```bash
   cmd.exe /c npm run dev
   ```

2. **Access the Preview**
   Once the console shows that the server is ready, navigate to:
   ```
   http://localhost:9002
   ```

3. **Verify Functionality**
   Verify that the login screen renders and you can interact with the app.
